# 📌 Overview

This project is a complete **E-Commerce Sales Analytics Solution** designed to transform raw sales data into meaningful business insights through:

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Advanced Analytics
- Customer Segmentation (RFM)
- Product Clustering
- Revenue Forecasting
- Interactive Dashboard Visualization

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/ANGEL-MAVUYANGWA/E-Commerce-Sales-Analytics-.git

cd E-Commerce-Sales-Analytics-
```

## Create Virtual Environment

### Windows
```bash
python -m venv .venv
.venv\Scripts\activate
```

### macOS/Linux
```bash
python3 -m venv .venv
source .venv/bin/activate
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

# ▶️ Running The Project

## Run Data Cleaning

```bash
python scripts/data_cleaning.py
```

## Run Data Analysis

```bash
python scripts/data_analysis.py
```

## Generate Visualizations

```bash
python scripts/visualization.py
```

## Launch Dashboard

```bash
streamlit run dashboard/app.py
```
